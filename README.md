# Webpack-Setup
The objective of the project is to use webpack to bundle javascript. 

# Getting started
- Follow the instructions from the [getting started guide](https://webpack.js.org/concepts/) to set up the basics. 
- Implement all the steps from Basic Setup to NPM Scripts.

**Just fork it and fell free to use it.**

To get a local copy up and running follow these simple example steps.

### Prerequisites

Having a web browser, code editor and git.

### Install

Just fork it and you are good to go.
